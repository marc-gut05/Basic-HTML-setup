Metadata

Basic data related to the webpage that is important to character encoding, authorship, search engines, browser cosmetics, and rich communication.

Character Encoding

A character encoding is another way of saying the character set that the document is permitted to use. This is specified so that the browser can properly display all the characters in the written elements. For example, with character encoding utf-8 the browser can properly display most languages in their proper form. However, with encoding ISO-8859-1, which is the encoding to display the latin alphabet, japanese characters will not be displayed.
